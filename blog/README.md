# How to post blogs

1. Add New blog under blog/posts in the form of date-title.html (2025-12-09-buildWebsite.html)

2. Add this blog to the blog/blog-manifest.json in the following form: 

   ```json
   {
     "id": "unique-post-id",
     "title": "Your Post Title",
     "date": "Month Day, Year",
     "readTime": "X minutes",
     "filename": "your-post-filename.html",
     "tags": ["Tag1", "Tag2", "Tag3"],
     "summary": "Brief summary of the post",
     "featured": true
   }
   ```
